# Campi

This is an app about camping related

## Stack and Tech

- Vite + React
- Capacitor
- Ionic
- Postgres + postgis (On supabase)
- React Query (Tanstack)
- React-Hook-Form + zod
- Leaflet
- LocationIQ - Geocoding and Reverse Geocoding
- Infisical - managing environments secrets
- Appflow - deployment, CI/CD
